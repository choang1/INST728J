4. Describe basic parts of a table and tags used to identify them?

A table is composed with the following tags: < table > table element, < tr > table row, < th > table header, and < td > table data.
In creating a table, an html file must lead and end with the < table > element and within it, a series of < tr > table rows.
Nested within the < tr > tags, one can denote a space as < th > table header (which will format similarly to a header tag) or < td > table data.
The more < td > you include within a < tr > will determine the number of columns a table will have.

5. How do you create cells that span more than one column or row?

One can create multiple rows within a table by implementing multiple sets of table row tags < tr >. Within a table row, the number of nested table data < td > sets determines the number of columns within that row.
For example, a table row with 3 nested < td > components will form 3 columns.

table row 1, column 1 header table row 1, column 2 header table row 1, column 3 header
table row 2, column 1 data table row 2, column 2 data table row 2, column 3 data
table row 3, column 1 data table row 3, column 2 data table row 3, column 3 data

7. Write down styling attributes you can use while creating table and explain what they do.

Presented in the unordered list below are some common attributes that can manipulate the presentation of a table.

  • < style > table, th, td
    • border : 1px solid black displays table border type; in this case it is a solid black border at 1 pixel (no border by default)
    • padding : 5px implements spacing within cells around text at 5 pixels.
    • text-align: left displays text alignment within table cells; left in this example.
    • background-color: lightyellow adds background color
  • col width="180px" manipulates width size of column at 180 pixels.
